FAQ

Does Lights Out force shutdown?

Only when the user explicitly chooses a force-shutdown path. The normal workflow keeps shutdown controls visible and user-controlled.

Does it track my sleep?

No. It tracks app events such as timer sessions and streaks. This is not medical sleep tracking.

Does remote control run by default?

No. Remote control is off by default and requires a pairing token when enabled.

Is there a portable version?

Yes. Releases include an installer and portable build.

How do I verify the download?

Use the release checksum file, SHA256SUMS.txt, published with the release assets.
