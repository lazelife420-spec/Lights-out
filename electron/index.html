🌅 First Light

Works with Sleep or Hibernate: schedules a Windows wake task, then runs a 60-second sunrise brightening sequence on resume. The morning bookend to your Last Light ritual.

🛡️ WiFi Guard

Block internet for kids' devices at bedtime.

Firewall blocks by IP. Router blocks by MAC (works for phones/tablets too). Adapter disables your WiFi entirely.
Click devices to add them. Router mode uses MAC addresses; Firewall mode uses IPs.
